# World — Isle of Ninefold

## Setting

A small island paradise in a warm sea — think Cycladic light meeting Jiangnan soft greens.
Whitewashed docks, painted boats, stone paths, and a central grove where schools leave
offerings without arguing… most days.

## Mood references

- *Wind Waker* sea horizon and cheerful NPC cadence  
- *Link's Awakening* "everyone knows everyone" density  
- Painted scrolls + fresco fragments as environmental storytelling  

## Landmarks

| Landmark | Role |
|----------|------|
| **Agora Grove** | Hub, save shrine, ferry posts, notebook UI |
| **Nine Districts** | Biomes for each school |
| **Tide Market** | Item shop, rumors, side quests |
| **Wind Shrine** | Music motif / later instrument tool |
| **Wreck Cove** | Tutorial beach where the Seeker wakes |

## Creatures (folksy)

| Creature | Behavior |
|----------|----------|
| **Shade-mite** | Soft enemy; pops into ink when hit |
| **Lantern crab** | Neutral; drops shell currency |
| **Wind-kites** | Ambient; lead toward secrets |
| **Stone tutors** | Immobile dialogue stones |

## Day cycle (later)

Dawn / noon / dusk / night tint; some mentors only speak at dusk.
